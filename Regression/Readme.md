# Прогнозирование стоимости недвижимости

Краткое описание проекта: решение задачи регрессии для прогнозирования стоимости жилых домов с использованием нескольких моделей машинного обучения.

## Задача проекта
Цель — построить модель, которая прогнозирует стоимость домов на основании большого количества фичей.

**Метрика:** коэф. детерминации R²

## Этапы работы
1.  **Препроцессинг:** Поиск пропусков и выбросов, кодирование категориальных переменных (кроме CatBoost), оценка корреляции метрик между собой.
2.  **Разбиение данных:** Разделение на обучающую (train) и тестовую (test) выборки.
3.  **Обучение моделей:** Обучение нескольких классических моделей для получения baseline.
4.  **Подбор гиперпараметров:** Оптимизация ключевых параметров с помощью GridSearchCV.

## Использованные модели
В проекте были обучены и сравнены следующие модели:
*   RandomForestRegressor
*   CatBoostRegressor
*   XGBRegressor

## Выводы
Наилучшие результаты показала модель XGBRegressor (R² = 0.919). CatBoostRegressor почти не уступает (R² = 0.912)
У RandomForestRegressor показатели немного ниже (R² = 0.896).
