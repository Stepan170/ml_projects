# Кредитный скоринг

Краткое описание проекта: решение задачи бинарной классификации для прогнозирования невозврата долга по кредиту с использованием нескольких моделей машинного обучения.

## Задача проекта
Цель — построить модель, которая прогнозирует, выплатит ли клиент долг по кредиту.

## Этапы работы
1.  **Препроцессинг:** Обработка пропусков, масштабирование числовых признаков для логистической регрессии, оценка корреляции метрик.
2.  **Разбиение данных:** Разделение на обучающую (train) и тестовую (test) выборки.
3.  **Обучение моделей:** Обучение нескольких классических моделей для получения baseline на 20% от всех данных.
4.  **Подбор гиперпараметров:** Оптимизация ключевых параметров с помощью GridSearchCV.

## Использованные модели
В проекте были обучены и сравнены следующие модели из библиотеки scikit-learn и xgboost:
