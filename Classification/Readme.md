# Прогнозирование оттока клиентов

Краткое описание проекта: решение задачи бинарной классификации для прогнозирования ухода клиентов с использованием нескольких моделей машинного обучения.

## Задача проекта
Цель — построить модель, которая прогнозирует, собирается ли клиент прекратить пользоваться услугами банка. Это позволяет бизнесу вовремя принимать меры по удержанию.

**Метрики:** precision, recall, F1_score, Roc-AUC. Для выбора лучшей модели - Roc-AUC

## Этапы работы
1.  **Препроцессинг:** Поиск пропусков, масштабирование числовых признаков для логистической регрессии, оценка корреляции метрик.
2.  **Разбиение данных:** Разделение на обучающую (train) и тестовую (test) выборки.
3.  **Обучение моделей:** Обучение нескольких классических моделей для получения baseline.
4.  **Подбор гиперпараметров:** Оптимизация ключевых параметров с помощью GridSearchCV.

## Использованные модели
В проекте были обучены и сравнены следующие модели из библиотеки scikit-learn и xgboost:
*   DecisionTreeClassifier
*   RandomForestClassifier
*   LogisticRegression
*   XGBClassifier

##   Сравнение результатов
Сравнение производительности моделей на тестовой выборке см. в конце bank.ipynb.

## Выводы
Наилучшие результаты по Roc-AUC показала модель XGBClassifier (0.938), но разница с RandomForestClassifier (0.935) незначительна.
Наихудший результат у LogisticRegression (0.769). Это значит, что ансамблевые модели лучше подходят в данной ситуации.
